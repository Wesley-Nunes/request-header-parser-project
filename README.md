## <a name="requirements"></a>Requirements

  - Node: >=20.0.0 <21

## How to Use

1. **Install the packages:**
   ```sh
   npm install
   ```
   
2. **Initialize the server:**
   ```sh
   npm run start
   ```

3. **Make a GET request to** `http://localhost:{env.PORT}/api/whoami`
